bash

shell: 리눅스 커널과 사용자를 연결해주는 인터페이스
- 사용자의 입력을 읽고, 해석하고, 프로그램을 실행함
- "셸 스크립트": 일련의 명령행을 한 파일에 담아놓은 것
- 리눅스 운영체제의 표준한 bash(bourne again shell)

문법
<프로그램명> <내용>
echo: 셸 명령, 내장명령, 프로그램
- 화면에 출력하는 명령
- " "사용가능, " "없이 변수지정 할 시 space없어야됨
- 변수를 사용할 때에는 '$'기호 사용한다
- 내장된 변수도 존재한다. 이것들은 대문자로 지정되어 있다

명령의 특징
- 한 프로그램이 한 가지 일만 한다
- 명령의 조합을 잘 해야 한다

명령
- 프로그램 자체
- 단순명령( simple command ) : 프로그램 + 내용
- 결합 명령( combined command ) : 프로그램 + 옵션+ 내용
- 옵션: 프로그램을 구체화하는 역할, - h

- ls : list

파이프
- |
- 명령행 | 명령행
- less
- 이전 명령행의 출력값을 다음 명령행의 입력값으로 
ex) 'echo PATH | less'

- wc, head, cut, grep, sort, unlq

입력
- stdin: standard input/in, 표준 입력

출력
- stdout: standard output/out, 표준 출력
- 출력 스트림

Q> /bin 내용을 라인별로 출력하라
--> ls -l /bin


1. wc
-파일 내용의 줄(line) 수, 단어(word) 수, 문자(charcater) 수 출력하는 명령

- wc 로 이 파일의 내용 출력하기
- Q> 지금 디렉터리의 하위 디렉터리 내용의 줄수, 단어수, 문자 출력하라.
--> ls | wc

명령인자
- w
- l
- c
txt 단어 몇 글자인지 세어보자
/bin 파일+디렉토리 세어보자

2. head
- 파일의 첫째줄 내용을 화면에 출력하는 방법
-n: 
  head -n3(default=10)
  - 파일이 90줄 -n100? --> 파일 다 출력

  ex1. 이 파일의 1번째 줄 단어수, 글자 수 세기
  ex2. /bin 파일 중에서 앞에 5개만 출력하기

3. cut
- 파일의 내용 중에서 특정 열만 출력하는 명령
- 범위 지정 방법:
- \t : -  f
James \t 34
f1        f2

cut -f1 name.txt

-3, 5, 6
cut -f3,5,6
- 4-5
  cut -f4-5

이름\t나이\t도시

Q1>3번째 필드 조회
Q2>1번째 필드 조회하는데 1번째 줄만 출력하기

-c : chracter
-c5
-c-5
-c1,5,6

# animal.txt 1필드의 마지막 글자 출력하기
=> 마지막 가져오기는 음수 인덱스로 접근하지 않는다.
